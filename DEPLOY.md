## 手动部署步骤（5分钟）

### 1. 打开GitHub创建仓库

浏览器访问：https://github.com/new

填写：
- **Repository name**: `ai-tool-lib`
- **Description**: `AI神器库 - 让每个普通人都能找到最适合自己的AI`
- **Public**: ✅ 勾选
- **Add a README file**: ❌ 不勾选
- **Add .gitignore**: ❌ 不勾选

点击 **Create repository**

### 2. 推送代码

在创建的仓库页面，向下滑动找到 "…or push an existing repository from the command line"

点击 **"Choose a file to upload"**，然后：
- 把 `C:\Users\86139\ai-tool-deploy\index.html` 文件拖进去
- 填写 Commit message: "Initial commit: AI神器库"
- 点击 **Commit changes**

### 3. 开启GitHub Pages

1. 点击仓库页面的 **Settings**
2. 左边找到 **Pages**
3. Build and deployment:
   - Source: 选择 **"Deploy from a branch"**
   - Branch: 选择 **"main"** (如果没有，点Save刷新)
4. 等待1-2分钟，页面会显示链接，如：
   - `https://wangye2022.github.io/ai-tool-lib/`
